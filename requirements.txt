# Troubleshooting Common Issues:
# - dlib compilation errors: Increase swap space or use pre-built wheels
# - OpenCV issues: sudo apt install python3-opencv
# - Serial permission errors: sudo usermod -a -G dialout $USER (reboot required)
# - Camera issues: sudo raspi-config -> Interface Options -> Camera -> Enable

# Performance Recommendations:
# - Use --no-cache-dir flag for pip install on limited storage
# - Consider using --find-links for local wheel packages
# - Set MAKEFLAGS="-j1" for single-core compilation on limited RAM

# - Raspberry Pi 4: Recommended for full feature set
